# LinkedIn-Clone
 
### This is a clone of linkedin built with _React, Redux, Firebase, and MaterialUI_. ###

## Features ##

* Login, Account-Creation, and Logout with Firebase authentication
* Firestore Database to store posts
* Create a new post
* Realtime update for posts
